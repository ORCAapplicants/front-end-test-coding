A continuación se enumera los pasos a seguir para instalar y correr
el proyecto desde la terminal:

1) npm run yarn //hay veces que es necesario correr esta linea con permisos de administrador.

2) yarn install

3) yarn gcp-global

4) ir a un navegador e introducir la dirección http://localhost:8080
